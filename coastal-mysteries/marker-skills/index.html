Marker Skills

Use Drawing Tools to create and modify markers. We are going to put a marker on the town of Tenby so that we can see it at any scale.

You will see that Drawing tools change depending on which tool is active. They also change if you select one or more drawn objects.

  1. Create a marker.
  2. Zoom out and back in again to see the marker at different scales.
  3. Select the marker you added by clicking on it. How does it change?
  4. What extra drawing tools appear?
  5. Work out how to delete a single marker?
  6. How about a cluster of markers?

When you are done, make sure to leave a marker at Tenby.
