Start a Spreadsheet

To start using a spreadsheet, log in to Purple Mash and launch 2Calculate in formula mode.

You can see that the app shows a grid of cells on the screen. You can select a cell by clicking with the pointer or by using the arrow keys to move around. You can enter letters and numbers into a the selected cell by typing or using the tool panel on the right.

  • Move around the grid in different ways: click or tap, arrow keys.
  • Type into a cell, what happens?
  • Can you change the colour of a cell?
  • What if you click or tap on the row or column edges?

At the end of your lesson, save your work in your own Purple Mash Work folder.
