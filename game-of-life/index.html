Conway's Game of Life

John Horton Conway (1937-2020) was a British mathematician. He devised this game in the 1970’s. The game plays out on a two dimensional grid, like an infinite chessboard. It is a zero player game because, once the board is set, there doesn’t need to be any more input from a human.

The starting board can even be set up in a random state, meaning no human decisions are ever needed. You would be forgiven for thinking that such a game would be uninteresting. How wrong you would be!
