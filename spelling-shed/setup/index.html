Setup

At the start of each academic year, there are a number of setup processes to run through. Some of them are handled centrally by the admin team (me and Emma), some you have agency over.

  1. Remove left pupils (e.g. Y6 leavers) and setup new classes. Central. Done.
  2. Check usernames and passwords and print out tickets for reading records. Central. Done though not without pain!
  3. Set the spelling level (i.e. year group) for your class. Adjust for children working below.
  4. Assign the spelling stage (year) to your class. You can set the whole year in one go. This will space out the steps week by week. You can then adjust what’s been done. Alternatively, you can set single assignments one-by-one.
